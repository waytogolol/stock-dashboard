## 方法論/統計

| 術語 | 白話 |
|---|---|
| **k10/k20/k60** | 進場後第10/20/60個交易日的累積報酬（前瞻窗） |
| **T+1開盤版** | 訊號日隔天開盤才進場的可交易性檢驗（排除「收盤才知道訊號」的作弊） |
| **rel / relo** | 扣掉同事件中位數的相對報酬（配對層,判真偽）；絕對報酬判上不上板 |
| **dd250 / dd52** | 距250日高（52週高）的回撤%＝位階 |
| **episode去重** | 連續觸發日只算第一天（隔≥10交易日才算新事件） |
| **bootstrap** | 重抽樣檢定：報酬是不是運氣（CI含0＝不過） |
| **LOTO** | leave-one-theme-out＝逐一剔除題材看結論會不會翻 |
| **case-control** | 從結果反推（飆股事前有沒有特徵X）＝算基準率避免倖存者敘事 |
| **MDD** | 最大回撤（權益曲線從高點跌最多的那段） |
| **單利Σ/複利** | 單利=每筆固定1單位加總（比訊號品質）；複利=資金滾動（受並發/空手期影響） |
| **前視(look-ahead)** | 用了當下不可能知道的未來資訊＝作弊；「靜態題材名單套歷史」是常見型 |
| **存活偏差** | 研究池只有活到今天的股票（下市的不在）＝報酬高估來源之一 |
| **股名帶星號（國巨\*）** | 減資註記——名單裡股名後的「*」＝辦理過減資（全市場約41檔）；程式對照公司名須剝除星號，否則整段歷史靜默漏抓（2026-07-20國巨解質事故根因） |
